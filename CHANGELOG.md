# Changelog

All notable changes to Starcat Safari Plugin are documented here.

## 0.1.0 - Unreleased

- Add GitHub repository page panel.
- Add Starcat local API client for `/plugin/v1`.
- Add Options page for local port and Local API Key pairing.
- Add Safari toolbar popup for quick local port and Local API Key pairing.
- Add support for recommendations, wiki links, private notes, health scores, OpenSSF scores, CodeFlow, and Codebase actions.
- Fix "Open in Starcat app" entry in the GitHub Code menu so its icon size and text baseline align with the surrounding native entries (e.g. "Open in GitHub Copilot app", "Download ZIP").
- Rename repository sidebar section titles to "Recommends", "Notes", and "Tags" and prepend the Starcat icon to each, so the sections read consistently with other Starcat entry points (README tab, Code menu).
