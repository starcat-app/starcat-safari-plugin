Setup

How to use

  1. Open Starcat and enable Browser Plugin Service.
  2. Copy the service URL and bearer token into this page.
  3. Click Test Connection and confirm the Starcat app responds.
  4. Open a GitHub repository page and refresh it once.

Troubleshooting

Quick checks

Starcat must be running, and the Browser Plugin Service must be enabled.

The extension must be reloaded after local plugin files change.

GitHub pages may need a hard refresh after changing the service URL or token.

Requests go to 127.0.0.1; firewall or proxy tools can block them.
